([build_case_oa.py](../build_case_oa.py), [nepi.py](../nepi.py), the models, or
the Atlas). See [ROADMAP.md](../../ROADMAP.md).

| Script | Why archived |
| ------ | ------------ |
| `basket_index_oa.py` | Illustrative "basket index" cross-check (retired 2026-06-09). Its CSVs/figures feed neither the paper's headline result nor the Atlas; the Access surface uses the empirical OLS penalty (`access_penalty_model.py`), not basket assumptions. |
| `basket_index_v1.py`, `build_case.py`, `diagnostic_fig1b.py`, `lsoa_figures.py`, `proof_of_concept_lsoa.py` | LSOA-era (v0) analysis, superseded by the OA-level analysis (~130 households/OA vs ~700/LSOA). |
